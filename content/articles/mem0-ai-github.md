### Basic Usage Example

Integrating Mem0 AI into an agent's workflow involves initializing the memory system and then using its methods to add and retrieve information. This is a fundamental step for implementing **AI memory** effectively.

```python
## Hypothetical code example for using Mem0 AI
## This example assumes a local ChromaDB is set up.
## Please refer to the Mem0 AI GitHub for specific configuration details.

from mem0_ai import Mem0

## Initialize Mem0 AI with a local ChromaDB backend
## The path specifies where ChromaDB will store its data.
memory = Mem0(
 vector_db_config={
 "type": "chroma",
 "path": "./mem0_chroma_db" # Directory for ChromaDB files
 }
)

print("Mem0 AI initialized with ChromaDB.")

## Add information to memory
memory.add(
 "User asked about the weather in London.",
 context="User query: What's the weather like today?"
)
print("Added memory: 'User asked about the weather in London.'")

memory.add(
 "The weather in London is currently sunny with a temperature of 20°C.",
 context="Weather API response for London"
)
print("Added memory: 'The weather in London is currently sunny...'")

## Retrieve relevant memories
search_query = "What is the current weather in London?"
print(f"\nSearching for memories related to: '{search_query}'")
retrieved_memories = memory.search(search_query)

print("\nRetrieved Memories:")
for mem in retrieved_memories:
 print(f"- {mem}")
## Expected output might be a list of relevant memory entries, e.g.,
## - The weather in London is currently sunny with a temperature of 20°C.
```
